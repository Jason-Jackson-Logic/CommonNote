# 更新日志

本项目的所有重要变更都将记录在此文件中。

## [2.0.0] - 2026-02-21

### 新增

#### 功能
- **笔记模板** - 6 个预设模板（空白、日志、会议记录、阅读笔记、项目文档、教程）
- **双向链接** - 使用 `[[笔记标题]]` 语法链接笔记，支持反向链接显示
- **图片上传** - 在编辑器中粘贴/拖放图片，自动上传到服务器
- **目录** - 从笔记查看器中的标题自动生成目录
- **Markdown 导入** - 批量导入 .md 文件，自动提取标题/标签
- **批量操作** - 多选模式，批量删除和导出

#### 性能
- **数据库索引** - 8 个索引用于优化查询
- **React Query** - 智能缓存，减少重复请求
- **分页 API** - 后端分页配合前端组件

### 修改
- 重构所有钩子函数以使用 React Query
- 改进编辑器用户体验，添加图片上传指示器
- 增强查看器，添加目录侧边栏

### 修复
- 数据库查询性能问题
